## Odoo AI CRM Note Analyzer

This custom Odoo module adds AI-powered analysis capabilities
to CRM customer notes.

### Features
- Custom CRM model with AI-related fields
- Python-based business logic
- SQL-backed ORM integration
- AI-based summarization and sentiment analysis (mock logic)

### Technologies
- Python
- Odoo ERP
- PostgreSQL

### Purpose
Designed to support digital transformation and intelligent
decision-making within ERP systems.
